# Quick Start Guide

## For End Users

### 1. Install APK on Android TV

- Build APK using Android Studio (see `BUILD_APK.md`)
- Install on your Android TV via USB, network, or file manager

### 2. Launch App

- Open "Screen Mirror" app on Android TV
- App automatically starts waiting for Mac connection
- Note the IP address displayed

### 3. Connect from Mac (No Installation!)

**On your Mac:**
1. Click AirPlay icon in menu bar (top right)
2. OR Press `Control + F2`
3. OR Go to System Settings → Displays → AirPlay Display
4. Select "Android TV" from the list
5. Your Mac screen will mirror to TV!

## Key Features

✅ **Zero Mac Installation** - Uses built-in Screen Mirroring  
✅ **Automatic Discovery** - TV appears in Mac's AirPlay menu  
✅ **4K Support** - High-quality mirroring  
✅ **Real-time Stats** - Network speed and FPS  

## Requirements

- Android TV with Android 7.0+
- Mac with Screen Mirroring support
- Both devices on same WiFi network
- Good WiFi signal (25+ Mbps recommended)

## Troubleshooting

**TV not appearing?**
- Check both devices on same WiFi
- Restart the app on TV
- Check Mac firewall settings

**Connection fails?**
- Check network speed (shown on TV)
- Ensure strong WiFi signal
- Restart both devices

For detailed instructions, see `INSTALL_INSTRUCTIONS.md`
